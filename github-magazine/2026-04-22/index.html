AI Agent Framework
The Agent That Teaches Itself
NousResearch/hermes-agent
A self-evolving AI agent with a closed learning loop. It creates skills from experience, refines them in use, searches its own conversations for context, and builds an ever-deepening model of who you are across sessions. V0.8.0 brought browser-use integration, remote backends, and worktree parallelism.
65.9K
GitHub Stars
Python
How to use this
Think of Hermes Agent as a persistent teammate that accumulates institutional knowledge. Deploy it to handle recurring automations that currently live in your head. The multi-platform gateway (Slack, Telegram, Discord) makes it a natural orchestrator for cross-tool workflows you already run manually.
02
AI Memory
Verbatim Recall for Machines
MemPalace/mempalace
An AI memory system built around verbatim storage rather than lossy summarization. MemPalace preserves exact context across sessions, giving language models a durable, searchable long-term memory that doesn't decay or hallucinate details. Crossed 20K stars within 48 hours of launch.
43.3K
GitHub Stars
Python
How to use this
Pair MemPalace with your existing agent setup to give it perfect recall of past decisions. Particularly valuable for automations that span multiple days or contexts, where standard context windows lose important details between sessions.

Developer Tools
Every Document Becomes Markdown
microsoft/markitdown
Microsoft's converter that turns PDFs, Word docs, Excel sheets, PowerPoints, images, and more into clean Markdown. The universal bridge between messy enterprise documents and the structured text that AI pipelines actually consume.
104.5K
GitHub Stars
Python
How to use this
Drop this into your document processing pipeline as the first step. If you're building RAG systems or feeding docs to language models, Markitdown solves the "garbage in" problem. Combine with your SEO or content workflows to ingest competitor research at scale.

AI Infrastructure
One Gateway, Every Model
QuantumNous/new-api
A unified AI model hub that cross-converts LLMs into OpenAI-compatible, Claude-compatible, or Gemini-compatible formats. Acts as a centralized gateway for personal and enterprise model management, with load balancing, key rotation, and usage tracking built in.
28.3K
GitHub Stars
Go
How to use this
If you're calling multiple LLM providers, this eliminates the adapter code. Self-host it as a proxy between your automations and model providers. Particularly useful if you're swapping between Claude, GPT, and Gemini depending on the task and want one consistent interface.
05
AI Development
Karpathy's Rules for Coding Agents
forrestchang/andrej-karpathy-skills
A single CLAUDE.md file that reshapes Claude Code's behavior around Andrej Karpathy's running observations on how large language models fail, guess, and recover. Drop it into a project before refactoring and watch the agent code with more discipline.
16.5K
GitHub Stars
Markdown
How to use this
Copy the CLAUDE.md directly into your project roots. It's most valuable during refactors and complex multi-file changes where LLMs tend to over-abstract or silently break contracts. Think of it as a behavioral guardrail, not a replacement for your own project instructions.

Multi-Agent Systems
Agents That Coordinate Like a Team
multica-ai/multica
A multi-agent coordination system purpose-built for coding teams. Multica orchestrates multiple AI agents to work on different parts of a codebase simultaneously, handling task decomposition, conflict resolution, and merge coordination automatically.
9.2K
GitHub Stars
TypeScript
How to use this
Think about the bottleneck in your current agent workflow: one agent, one file at a time. Multica lets you parallelize complex refactors across a codebase. Start with well-scoped tasks like migrating test files or updating API signatures across multiple services.
09
Voice AI
70 Milliseconds to a Conversation
NVIDIA/personaplex
Full-duplex voice AI from NVIDIA achieving 0.07-second latency. Supports simultaneous listen-and-speak, making natural voice conversations with AI feel genuinely real-time for the first time. Built for production voice agent deployments.
9.0K
GitHub Stars
Python
How to use this
If you're building any voice-driven experience, the 70ms latency benchmark is the new bar. Evaluate PersonaPlex as the speech layer for customer-facing agents. The full-duplex capability matters most for sales and support bots where interruption handling separates good from unusable.
10
Speech Synthesis
30 Languages, No Tokenizer
OpenBMB/VoxCPM
A tokenizer-free, diffusion autoregressive TTS model with 2B parameters. VoxCPM2 supports 30 languages at 48kHz studio quality, trained on over 2 million hours of multilingual speech. Design novel voices from text descriptions or clone any voice from a short clip.
7.0K
GitHub Stars
Python
How to use this
The voice-from-description feature is the standout. Instead of curating voice actor libraries, describe the voice you want in natural language and generate it. Ideal for prototyping multilingual content or building voice experiences where licensing real voice talent isn't feasible yet.

Game Development

5.2K
GitHub Stars

How to use this
Even if you're not making games, study the agent hierarchy pattern. The way it decomposes a complex creative workflow into coordinated specialist roles is a template for any multi-agent system. Borrow the coordination architecture for your own domain-specific agent teams.
